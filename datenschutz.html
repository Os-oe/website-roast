2. Die eingegebene Website-Adresse („Rösten!")

Wenn du eine URL eingibst und auf „Rösten!" klickst, wird diese Adresse an unseren Server übermittelt und dort wie folgt verarbeitet:

  • Erzeugung der Röstung: Der ausgelesene Seiteninhalt wird zusammen mit unserer Aufgabenbeschreibung an die Gemini-API von Google (Google LLC bzw. Google Ireland Ltd.) übermittelt, die daraus die humoristische „Röstung" (Score, Sprüche, Tipp) erzeugt. Dabei kann eine Übermittlung in die USA stattfinden (abgesichert über das EU-U.S. Data Privacy Framework bzw. Standardvertragsklauseln).

Eine Speicherung der Röstungen oder URLs auf unserem Server findet nicht statt.
